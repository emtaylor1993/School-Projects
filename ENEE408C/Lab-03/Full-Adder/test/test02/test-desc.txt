Test 02: Full Adder

This test will validate the functionality of the full adder using the following
inputs of the format (Carry-in, A, B):

(1, 0, 0)
(0, 1, 0)
(1, 0, 1)
(1, 1, 1)

This test uses the diff command to test the results by creating an empty
correct-output.txt file and comparing the out.txt file against the 
correct-out1.txt file using the diff command.
